# Vyra

a menu bar app (also show a proper app if opened) that has: 
1. todo list - personal/ projects
  - manual time logging
  - activity sessions timers/ pomodoro
3. gives a productivity score
  - productivityScore = focusTime * weight - idlePenalty - distractionPenalty
4. analytics - weekly/monthly - timeline format/ or some other format
  - category summaries wise
5. automatic idle detection

## Example Menubar (this should be fast)
    Vyra Menu
    -------------------
    Current Session
    Coding (25m)

    Start Pomodoro
    Add Task
    Start Timer

    Today
    Focus: 3h 20m
    Idle: 40m
    Score: 82
